Grammar Ambiguity Checker

A simple web-based tool to check for grammatical errors and ambiguities in text.

Features
- Checks for double words (e.g., "the the")
- Detects consecutive verbs (e.g., "I will go run")
- Identifies passive voice (e.g., "The ball was thrown")
- Provides suggestions for improvement

Technologies Used
- HTML5
- CSS3
- JavaScript

Getting Started
1. Clone the repository: `git clone https://github.com/your-username/grammar-ambiguity-checker.git`
2. Open `index.html` in a web browser
3. Enter your text in the textarea and click the "Check Grammar" button
